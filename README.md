# Lasia_ui

LasiaUI is a Neumorphism Style UI component library based on Vue 2

## Install

```shell
npm i lasia-ui
```

## Quick Start

```javaScript
// main.js
import Vue from 'vue'
import LasiaUI from 'lasia-ui'
import 'element-ui/lib/theme-chalk/index.css';

Vue.use(LasiaUI)
```

## Example

```Html
<Template>
    <ls-button round>Hi Lasia</ls-button>
</Template>
```
